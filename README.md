# Proyecto: Mi Perfil en Laravel
## Descripción
Aplicación web desarrollada en Laravel que muestra mi perfil personal usando un sistema de rutas y vistas Blade, con un diseño CSS 100% propio (Sin Bootstrap).

## Requisitos del sistema
- PHP 8.x
- Composer

## Instrucciones de instalación
1. Clonar este repositorio.
2. Ejecutar `composer install`.
3. Copiar archivo de entorno: `cp .env.example .env`
4. Generar llave: `php artisan key:generate`
5. Iniciar servidor local con `php artisan serve`.
